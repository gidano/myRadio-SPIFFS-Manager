## HU

### Leírás
A **myRadio SPIFFS Manager** egy egyszerű Windows alkalmazás, amely lehetővé teszi az ESP32 alapú myRadio eszköz SPIFFS fájlrendszerének kezelését soros kapcsolaton keresztül.

### Főbb funkciók
- Kapcsolódás COM porton keresztül
- SPIFFS fájllista lekérdezése
- Fájlok feltöltése (kijelölt mappába)
- Teljes mappa feltöltése
- Fájlok letöltése
- Fájlok törlése
- Teljes mentés (ZIP)
- Mentés visszaállítása
- Rádió újraindítása
- Két nyelv támogatása (HU / EN)
- Státuszsor részletes visszajelzéssel (pl. célmappa)

### Használat
1. Válaszd ki a COM portot
2. Kattints a **Kapcsolódás** gombra
3. Indítsd el a **Karbantartó módot**
4. Frissítsd a fájllistát
5. Válassz ki egy mappát (opcionális)
6. Használd a kívánt műveletet (feltöltés, törlés stb.)

## EN

### Description
**myRadio SPIFFS Manager** is a simple Windows application for managing the SPIFFS filesystem of an ESP32-based myRadio device over serial connection.

### Features
- Connect via COM port
- List SPIFFS files
- Upload files (to selected folder)
- Upload entire folders
- Download files
- Delete files
- Full backup (ZIP)
- Restore backup
- Reboot device
- Dual language support (HU / EN)
- Detailed status feedback (e.g. target folder)

### Usage
1. Select COM port
2. Click **Connect**
3. Start **Maintenance mode**
4. Refresh file list
5. Select a folder (optional)
6. Perform operations (upload, delete, etc.)
